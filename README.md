# DeepinWine_Debian_Buster

这是一个为Debian Buster x64重新打包的Deepin Wine，打包时间为2019-5-3，Deepin Wine相关版本为2.18-18~rc2

安装方法：
直接dpkg -i 安装目录下所有deb包，然后用apt install -f补全依赖，即可安装Deepin Wine移植的诸多应用。
